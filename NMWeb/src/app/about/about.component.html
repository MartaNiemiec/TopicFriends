About PeopleMatcher

This app connects people based on:
  • Actionable interests (e.g.: would like to do pair programming on ).
    • See list of topics which you can add to your User Profile
  • Geo-locations (see Users Map)
  • Meetings/meetups attendance (see Meetings List)

A community project.

Open-Source: github.com/karol-depka/NetworkingMatcher
